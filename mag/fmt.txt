FileMaker es una plataforma que le permite construir aplicaciones que se adaptan exactamente a lo que necesitan las empresas. Las ventajas con relación a las aplicaciones prefabricadas es enorme: en las aplicaciones personalizadas la curva de aprendizaje es practicamente inexistente. Estas aplicaciones pueden ir evolucionando en la medidad que la empresa o sus empleados lo necesiten.

Como plataforma, Filemaker es única.Combina un motor de base datos con herramientas de creación de interfaces que permiten trabajar con los datos que almacena, que pueden ser accesibles para cientos de usuariossimultáneamente, desde un móvil o un navegador. 

No existe ningún prerequisito para tomar este curso. En la comunidad de desarrolladores de Filemaker hay profesionales de todo tipo - desde médicos hasta arquitectos e ingenieros-, pero tambien estudiantes y autodidactas. El único requisito es tener curiosidad, la necesidad de encontrar una solución a las deficiencias del sistema de información que use o sencillamente aprender a construir cualquier aplicación desde cero.

La plataforma tiene 2 componentes esenciales:
la aplicación Filemaker Pro que nos permite construir aplicaciones de base de datos en Filemaker y 
Filemaker server: que es una aplicación de de servidor que nos permite compartir las aplicaciones construidas en Filemaker. Filemaker server permite acceder a la aplicación mediante cualquier navegador , mediante la app gratuita Filemaker Go ( sólo para iphones y ipads) o el mismo Filemaker Pro. El conjunto de tecnologías que nos permite acceder via web Filemaker lo denomina Webdirect. Nos presenta un interfaz practicamente idéntico al que vemos en la aplicación de escritorio sin necesidad de tener que construirlo en HTML, CSS, Javascript y PHP u otros lenguajes. Una vez alojado en el servidor, tambien permite compartir  los datos con otros sistemas mediante el protocolo estándar de bases de datos mediante su Data API.
