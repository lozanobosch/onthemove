File Maker es una plataforma que nos permite construir aplicaciones que se adaptan exactamente a lo que necesitamos. 
Las ventajas con relación a las aplicaciones prefabricadas es enorme.
En las aplicaciones personalizadas la curva de aprendizaje es prácticamente inexistente. Estas aplicaciones pueden ir evolucionando en la medida que la empresa o sus empleados lo necesiten.

Como plataforma, File Maker es única. Combina un motor de base de datos con herramientas de creación de interfaces. Esto nos permite crear, modificar o borrar la información y también crear informes complejos de todo tipo con textos, imágenes y gráficos. También nos permite exportar los datos en excel y csv. Cientos de usuarios pueden acceder a la aplicación resultante de forma simultánea, desde un móvil o un navegador web. 
File Maker tiene más de 30 años en el mercado y es, sin lugar a dudas, una sofisticada arma de digitalización masiva.
Es un un producto de Claris, una empresa subsidiaria de Apple.

No existe ningún prerequisito para aprender a construir aplicaciones con File Maker. En la comunidad de desarrolladores de File Maker, hay profesionales de todo tipo - desde médicos hasta arquitectos-, pero también estudiantes y autodidactas. Lo único que necesitas es tener curiosidad, la necesidad soluciónar u optimizar las deficiencias del sistema de información que utilizas  o sencillamente querer saber cómo construir cualquier aplicación desde cero.

La plataforma tiene 2 componentes esenciales.
La aplicación File Maker Pro que nos permite construir aplicaciones de base de datos.

Y File Maker server: que es una aplicación de servidor que nos permite acceder a la aplicaciones que construimos, mediante cualquier navegador, con la aplicación gratuita para dispositivos móviles o usando  File Maker Pro en Windows o Mac. 
El conjunto de tecnologías que nos permite acceder via web, Claris lo denomina Webdirect. 
El interfaz es prácticamente idéntico al que vemos en la aplicación de escritorio. 
Este interfaz web se construye sin usar una sóla línea de código en HTML, CSS o Lláva script, ni necesita ningún otro lenguaje de servidor. Es sencillamente increíble.

Una vez alojado en el servidor, también permite compartir  los datos con otros sistemas, mediante el protocolo estándar de bases de datos - llamado ODBC- o mediante su potente API de datos.

la versatilidad de Filemaker no termina aqui: los usuarios más avanzados usan Lláva Script en un componente web que tiene Filemaker, que permite crear interfaces sosfisticados y crear cuadros de mandos a la altura de las aplicaciones de business intelligence. Y a través de funcionalidades como cURL y plugins permite integrar aplicaciones externas de todo tipo: desde Paypal hasta  Machine learning. Las posibilidades de Filemaker sólo tienen el límite de nuestra imaginación.
